# E-commerce API Secure

API RESTful segura e robusta para lojas virtuais, com autenticação JWT e gerenciamento completo de produtos e pedidos.

## Tecnologias Utilizadas
- Node.js
- Express.js
- MongoDB
- JWT (JSON Web Tokens)

## Funcionalidades
- Cadastro e login de usuários
- CRUD de produtos
- Gerenciamento de pedidos
- Autenticação e autorização seguras
